# swrDelta

{/* 이 문서의 내용은 app 라우터와 pages 라우터 사이에 공유됩니다. Pages 라우터에만 특정한 내용을 추가하려면 `<PagesOnly>Content</PagesOnly>` 컴포넌트를 사용할 수 있습니다. 공유된 내용은 컴포넌트로 감싸지 않아야 합니다. */}

ISR이 활성화된 페이지에 대해 CDN이 `Cache-Control` 헤더에서 사용할 사용자 정의 stale-while-revalidate 기간을 지정할 수 있습니다.

`next.config.js`를 열고 `swrDelta` 설정을 추가하세요:

```js filename="next.config.js"
module.exports = {
  // 1년을 초 단위로 표현
  swrDelta: 31536000,
};
```

이제 `Cache-Control` 헤더에 빈 `stale-while-revalidate` 기간 대신 사용자 정의 기간이 포함됩니다.
