# typedRoutes

[정적으로 타입이 지정된 링크](/docs/app/building-your-application/configuring/typescript#statically-typed-links)에 대한 실험적 지원입니다. 이 기능을 사용하려면 프로젝트에서 App Router와 TypeScript를 함께 사용해야 합니다.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
};

module.exports = nextConfig;
```
