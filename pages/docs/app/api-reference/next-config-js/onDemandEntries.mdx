# onDemandEntries

{/* 이 문서의 내용은 app 라우터와 pages 라우터 간에 공유됩니다. `<PagesOnly>Content</PagesOnly>` 컴포넌트를 사용하여 Pages 라우터에만 특정한 내용을 추가할 수 있습니다. 공유된 내용은 컴포넌트로 감싸지 않아야 합니다. */}

Next.js는 개발 중 서버가 빌드된 페이지를 메모리에서 어떻게 처리하거나 유지할지에 대한 일부 제어 권한을 제공하는 옵션을 노출합니다.

기본값을 변경하려면 `next.config.js`를 열고 `onDemandEntries` 설정을 추가하세요:

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // 서버가 버퍼에 페이지를 유지할 기간 (ms)
    maxInactiveAge: 25 * 1000,
    // 동시에 처리되지 않고 유지되어야 하는 페이지 수
    pagesBufferLength: 2,
  },
};
```
