# generateEtags

{/* 이 문서의 내용은 app 라우터와 pages 라우터 간에 공유됩니다. Pages 라우터에만 특정한 내용을 추가하려면 `<PagesOnly>내용</PagesOnly>` 컴포넌트를 사용할 수 있습니다. 공유된 내용은 컴포넌트로 감싸지 않아야 합니다. */}

Next.js는 기본적으로 모든 페이지에 대해 [etag](https://en.wikipedia.org/wiki/HTTP_ETag)를 생성합니다. 캐시 전략에 따라 HTML 페이지에 대한 etag 생성을 비활성화하고 싶을 수 있습니다.

`next.config.js`를 열고 `generateEtags` 옵션을 비활성화하세요:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
};
```
