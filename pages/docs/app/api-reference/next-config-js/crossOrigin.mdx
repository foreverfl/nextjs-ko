# crossOrigin

{/* 이 문서의 내용은 app 라우터와 pages 라우터 간에 공유됩니다. <PagesOnly>내용</PagesOnly> 컴포넌트를 사용하여 Pages 라우터에만 해당하는 내용을 추가할 수 있습니다. 공유된 내용은 컴포넌트로 감싸지 않아야 합니다. */}

`crossOrigin` 옵션을 사용하여 [`next/script`](/docs/app/building-your-application/optimizing/scripts) 컴포넌트에 의해 생성된 모든 `<script>` 태그에 [`crossOrigin` 속성](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin)을 추가하고, 크로스 오리진 요청을 어떻게 처리해야 하는지 정의할 수 있습니다.

```js filename="next.config.js"
module.exports = {
  crossOrigin: "anonymous",
};
```

## 옵션

- `'anonymous'`: [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous) 속성을 추가합니다.
- `'use-credentials'`: [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials) 속성을 추가합니다.
