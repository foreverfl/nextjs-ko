# instrumentationHook

{/* 이 문서의 내용은 app 라우터와 pages 라우터 간에 공유됩니다. `<PagesOnly>Content</PagesOnly>` 컴포넌트를 사용하여 Pages 라우터에만 특정한 내용을 추가할 수 있습니다. 공유된 내용은 컴포넌트로 감싸지 않아야 합니다. */}

실험적인 `instrumentationHook` 옵션을 사용하면 Next.js 앱에서 [`instrumentation` 파일](/docs/app/api-reference/file-conventions/instrumentation)을 통해 모니터링을 설정할 수 있습니다.

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
};
```
