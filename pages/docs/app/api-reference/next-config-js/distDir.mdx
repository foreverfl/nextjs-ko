# distDir

{/* 이 문서의 내용은 app 라우터와 pages 라우터 간에 공유됩니다. <PagesOnly>내용</PagesOnly> 컴포넌트를 사용하여 Pages 라우터에만 해당하는 내용을 추가할 수 있습니다. 공유된 내용은 컴포넌트로 감싸지 않아야 합니다. */}

`.next` 대신 사용할 사용자 정의 빌드 디렉토리의 이름을 지정할 수 있습니다.

`next.config.js`를 열고 `distDir` 설정을 추가하세요:

```js filename="next.config.js"
module.exports = {
  distDir: "build",
};
```

이제 `next build`를 실행하면 Next.js는 기본 `.next` 폴더 대신 `build`를 사용할 것입니다.

> `distDir`은 프로젝트 디렉토리를 벗어나서는 **안 됩니다**. 예를 들어, `../build`는 **유효하지 않은** 디렉토리입니다.
