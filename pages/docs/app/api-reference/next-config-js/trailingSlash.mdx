# trailingSlash

{/* 이 문서의 내용은 app 라우터와 pages 라우터 사이에 공유됩니다. `<PagesOnly>Content</PagesOnly>` 컴포넌트를 사용하여 Pages 라우터에만 특정한 내용을 추가할 수 있습니다. 공유된 내용은 컴포넌트로 감싸지 않아야 합니다. */}

기본적으로 Next.js는 후행 슬래시가 있는 URL을 후행 슬래시가 없는 상대 URL로 리다이렉트합니다. 예를 들어, `/about/`은 `/about`으로 리다이렉트됩니다. 이 동작을 반대로 구성할 수 있어, 후행 슬래시가 없는 URL이 후행 슬래시가 있는 상대 URL로 리다이렉트되도록 할 수 있습니다.

`next.config.js`를 열고 `trailingSlash` 설정을 추가하세요:

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
};
```

이 옵션을 설정하면 `/about`과 같은 URL이 `/about/`로 리다이렉트됩니다.

[`output: "export"`](/docs/app/building-your-application/deploying/static-exports) 설정과 함께 사용할 때, `/about` 페이지는 (기본값인 `/about.html` 대신) `/about/index.html`로 출력됩니다.
