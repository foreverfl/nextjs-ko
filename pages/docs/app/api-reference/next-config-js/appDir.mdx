# appDir

> **알아두면 좋은 점**: 이 옵션은 Next.js 13.4 버전부터 **더 이상 필요하지 않습니다**. App Router는 이제 안정화되었습니다.

App Router([`app` 디렉토리](/docs/app/building-your-application/routing))는 [레이아웃](/docs/app/building-your-application/routing/layouts-and-templates), [서버 컴포넌트](/docs/app/building-your-application/rendering/server-components), [스트리밍](/docs/app/building-your-application/routing/loading-ui-and-streaming), 그리고 [데이터 패칭 동시 배치](/docs/app/building-your-application/data-fetching)를 지원합니다.

`app` 디렉토리를 사용하면 자동으로 [React Strict Mode](https://react.dev/reference/react/StrictMode)가 활성화됩니다. [`app`을 점진적으로 도입하는 방법](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app)에 대해 알아보세요.
