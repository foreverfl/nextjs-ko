# draftMode

`draftMode` 함수를 사용하면 [서버 컴포넌트](/docs/app/building-your-application/rendering/server-components) 내에서 [미리보기 모드(draftmode)](/docs/app/building-your-application/configuring/draft-mode)를 감지할 수 있습니다.

```jsx filename="app/page.js"
import { draftMode } from "next/headers";

export default function Page() {
  const { isEnabled } = draftMode();
  return (
    <main>
      <h1>My Blog Post</h1>
      <p>Draft Mode is currently {isEnabled ? "Enabled" : "Disabled"}</p>
    </main>
  );
}
```
