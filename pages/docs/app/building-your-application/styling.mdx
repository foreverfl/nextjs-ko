# 스타일링

Next.js는 애플리케이션 스타일링을 위한 다양한 방법을 지원합니다. 다음과 같은 방법이 있습니다:

- **CSS 모듈**: 이름 충돌을 방지하고 유지보수성을 향상시키기 위해 로컬로 범위가 지정된 CSS 클래스를 생성합니다.
- **전역 CSS**: 전통적인 CSS에 익숙한 사람들에게 사용하기 쉽고 친숙하지만, CSS 번들 크기가 커지고 애플리케이션이 성장함에 따라 스타일 관리가 어려워질 수 있습니다.
- **Tailwind CSS**: 유틸리티 클래스를 조합하여 빠르게 사용자 정의 디자인을 만들 수 있는 유틸리티 우선 CSS 프레임워크입니다.
- **Sass**: 변수, 중첩 규칙, 믹스인 등의 기능으로 CSS를 확장하는 인기 있는 CSS 전처리기입니다.
- **CSS-in-JS**: JavaScript 컴포넌트에 CSS를 직접 포함시켜 동적이고 범위가 지정된 스타일링을 가능하게 합니다.
